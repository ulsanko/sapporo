1. Sapporo_weather_daily_10Y.csv — 일별 기상

컬럼설명
date날짜 (YYYY-MM-DD)
precipitation_mm일 강수량 합계(mm). 없으면 0
temp_mean / temp_max / temp_min일 평균/최고/최저 기온(℃)
humidity_mean일 평균 상대습도(%)
wind_mean_ms일 평균 풍속(m/s)
sunshine_h일조시간(시간)
snowfall_cm / snow_depth_max_cm일 강설량 / 최심적설(cm). 없으면 0
  • 참고 기준: 真夏日 = temp_max ≥ 30, 猛暑日 = ≥ 35, 熱帯夜 = temp_min ≥ 25, 여름일 = temp_max ≥ 25.
  • 활용 예: 열대야·폭염 예측(분류/회귀), 열중증 데이터와 날짜로 조인해 "내일 위험" 모델 → 알림 시점 결정.

컬럼설명
date / time출몰 날짜·시각

latitude / longitude위도·경도 — 지도 시각화·공간 분석 가능
situation상황(목격·흔적·포획 등, 일본어 원문)
  • 삿포로 시내 한정(홋카이도 전체 아님). 2025년은 11월 말까지 361건으로 역대 최다.
  • 활용 예: 월·시간대·구별 위험도 분류 → 위험 구간에서 주의 모드 기본 켜기, 시간대 회피 권고.
  • 라이선스: CC-BY 4.0 — 앱·논문에 "출처: 札幌市" 표기 필요.

3. Hokkaido_heatstroke_daily_10Y.csv — 일별 열중증 이송

컬럼설명
date날짜 (조사 기간: 매년 5~9월만)
transported_total홋카이도 이송 인원 합계(명)
age_*연령 구분: 신생아/영유아/소년/성인/고령자/불명
sev_*중증도: 사망/중증/중등증/경증/기타
place_*발생 장소: 주거/직장①②/교육기관/공공(실내·실외)/도로/기타 — 2019년 이전은 빈칸
japan_total같은 날 전국 이송 합계(명) — 비교·맥락용
  • 홋카이도 전체 수치(삿포로만이 아님). 겨울 데이터는 없음(조사 자체가 5~9월).
  • 활용 예: 기상 데이터와 조인해 기온→이송 회귀, 고령자 비중 분석 → 폭염 안부 알림의 대상·시점·메시지 설계.

4. Japan_visitor_arrivals_monthly_10Y.csv — 방일 외객 월별·국가별

컬럼설명
year / month연·월
market국가·지역명(일본어). 주의: 総数·アジア計 같은 합계 행 포함 — 국가별 분석 시 제외할 것
arrivals입국자 수(명)
  • 전국 수치(홋카이도 한정 아님). 최근 2~3개월은 추계치. 2020~2022는 코로나 급감 구간 — 시계열 모델 학습 시 유의.
  • 활용 예: 계절성 분해·월별 예측, 국가 군집(여름형/겨울형 시장) → 혼잡 예측·분산 추천의 근거.

일본어 값 빠른 해석 (전처리용)

원본은 일본 공식 자료라 일부 값이 일본어다. 분석 전에 매핑하라. 형식 = 일본어 (발음) — 뜻.
